Overview

건설 그룹 HOBAN 사이트를 리디자인

저희 웹 페이지는 가독성을 높이기 위해 레이아웃 구조, 배치와 간격, 그리고 텍스트 자체도 읽기 쉽도록 주의깊게 디자인하였습니다. 또한, 독자들이 피로감을 느끼지 않도록 신경을 썼습니다.

모바일 기기에서도 웹 페이지를 원활하게 이용할 수 있도록 반응형으로 제작하였습니다.

Design

Concept

사용자들에게 편안하고 편리한 경험을 제공하기 위해 시각적 으로 직관적인 디자인을 적용한 웹사이트를 구축하였습니다.

Problems, Solution

디바이스 호환성을 위해 어떻게 했나요?

원하는 위치에 컨텐츠가 배치되지 않거나 계속해서 구조가 파괴되는 문제가 발생했지만, class 명이나 HTML 구조를 체계적으로 정리하여 충돌을 최소화 시킴

스크립트가 안되었을때의 문제?

네비게이션 및 페이징 요소를 지정하고 구성함으로써 문제를 해결

스와이퍼 이미지의 구현?

사용자가 직접 페이지를 넘기면 동작하지 않는 문제가 생겨, disableOnInteraction: false 값을 주어 문제를 해결

Review

공부하고 처음으로 반응형 웹사이트를 제작 했습니다.

설계할때 모바일, 태블릿, 웹에 대한 미디어 특성의 이해와, 그 외의 요소들을 구현하기 위해 animation, flex 등 요소의 배치에 대한 고민도 하고, 수정 및 보완 과정을 겪으면서 추후 작업에 더 많은 기술요소들을 활용 할 수 있을 것 같다.

Overview

수원 FC 사이트 리디자인

현재 축구 사이트 디자인이 구식으로 느껴지는 문제가 있어 리디자인을 통해 사용자들이 더 나은 경험을 얻도록 개선하고자 했다. 사용자들이 사이트를 방문하며 느끼는 편안함과 더 나은 시각적 인상을 줄 수 있도록 디자인 하였다.

Design

Concept

사용자들에게 시각적으로 즐거운 경험을 제공하고, 편의성을 높힌 디자인을 구현하였다.

Problem & Solution

문제 해결

커스텀 커서 활용하기

기본적인 마우스 커서 대신 사용자 정의 디자인으로 마우스 커서를 대체하기 위해 window.addEventListener를 활용해 마우스 움직임을 감지해 함수를 호출하게끔 하였다.

body.addEventListener를 활용해 특정 페이지를 벗어 날 때 마우스를 숨기는 효과를 주어 원하는 섹션에서 마우스 효과를 주었다.

hover시 크기는 변하지 않고 이미지가 확대대는 효과

부모 요소 크기를 고정하고 "overflow: hidden"을 사용하여 이미지에 "hover" 효과를 주면 돋보기로 확대한 듯한 효과를 줄 수 있었다.

동영상 + 커스텀 커스 충돌

addVideoEventListeners를 사용해 pointer-events를 임시로 꺼서 커스텀 커서와의 충돌을 방지하고자 하였다.

Review

부족한 점

스크롤을 했을 때 요소에 나타나기 애니메이션 효과를 구현해주는 라이브러리를 구현하고자 했는데, 높이 값의 인식 문제 때문에 계속 오류가 나서 결국, 애니메이션으로 대체 한 것 이 너무 아쉽다.

배운 점

사이트를 선택할때, 내 실력보다 좀 더 어려운 것에 도전하고싶어, 할 수 있을까 싶은 것을 골랐다. 모르는 건 찾아보고, 알아가면서 하다보니 처음 생각보다 좋은 디자인이 나온것 같았다.

잘한 점

가이드라인 제작, 퍼블리싱은 3일도 안걸렸던 것같다. 나머지 시간은 안되는 스크립트를 붙잡고 있었는데, 결국 구현은 못했지만 끝까지 구현하고싶어서 포기하지 않고 도전한 자세가 좋았던 것 같다.

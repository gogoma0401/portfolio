Overview

수원 FC 사이트 리디자인

현재 축구 사이트 디자인이 구식으로 느껴지는 문제가 있어 리디자인을 통해 사용자들이 더 나은 경험을 얻도록 개선하고자 했다. 사용자들이 사이트를 방문하며 느끼는 편안함과 더 나은 시각적 인상을 줄 수 있도록 디자인 하였다.

Design

Concept

사용자들에게 시각적으로 즐거운 경험을 제공하고, 편의성을 높힌 디자인을 구현하였다.

Keyword

#Motion Web Design

#사용자친화적

Typography

폰트는 Noto Sans Korean을 사용해 접근성을 높이고, 깔끔하게 표현하고자 했다.

Problem & Solution

문제 해결

메인 페이지 오픈시, 곰 얼굴을 형상화한 도형이 모였다가, 다시 흩어지게 하기!

웹페이지 로딩시 이미지를 반복적으로 가운데로 모이는 것을 표현하고 싶었는데, 모이고 다시 원래위치로 돌아지 않았다. 로딩상태를 제거하고 닫시 실행되게 해주었더니, 실행시 이미지들이 모이고 로딩 상태(일시 정지) 상태가 표현된후 반복적으로 이미지들이 원래 상태로 돌아게끔 구현하였다.

기술서 보러가기chevron_right

hover시 텍스트가 하나씩 나타나는 이벤트 효과

기본 상태에는 글자가 보이는데 hover시 각각의 텍스트가 하나씩 구현되게 해야 하였다. 트렌지션값을 조정하여 순차적으로 나타나게 하여 좀 더 임펙트 있게 적용 시킬 수 있었다.

각 페이지별 통일된 레이아웃, 요소 구축

웹페이지에서 각 페이지별로 통일성있게 작업해야 하기에, 스타일 가이드 제작을 하여 작업및 변수(var)사용으로 작업했으나, 부족하였다. git을 통한 활용과 html&Css에 주석을 첨가해 좀 더 안정적으로 구축할 수 있었다.

Review

부족한 점

잘 만든 디자인에 비해 JavaScript에 대한 이해도가 아직은 부족, 미숙해서 동작을 많이 구현하지 못해 아쉬웠다.

일러스트 디자인를 통해 좀 더 풍성한 효과나 다양한 동작들을 적용시키지 못한게 많이 아쉽다.

배운 점

협업하면서 프로젝트를 단순히 완성시키는 것 뿐 아니라, 그 과정속에서 부족한 부분을 채우고, 팀장이라는 책임감뿐만 아니라 ‘팀원 보다 더 많이 알아야 그 프로젝트를 이끌어 갈 수 있겠다’ 는 생각을 하게 되었습니다. 그리고 스크립트에 대한 이해와 중요성을 여실히 깨닫게 된 것 같습니다.

잘한 점

계획했던 가이드라인 제작 부분을 디자인과 퍼블리싱까지 동일하게 주어진 시간 내에 했다는 점에서 상당히 잘 이루워 진 것 같다.

일러스트 디자인를 통해 좀 더 풍성한 효과나 다양한 동작들을 적용시키지 못한게 많이 아쉽다.

마치며...

짧은 시간동안 팀 프로젝트를 하게되며 나의 부족한 점과 보완해야될 점을 알게 되었다. 배운 것을 바로 적용시켜 활용까지 한다는 점이 직접하면서도 어렵지만서도, 흥미로웠던 것 같다. 깃의 협업 기능을 통해 좀 더 편리하게, 효울적으로 작업하기도 했지만, 서로 작업중에 충돌하거나, 구현하기 힘든 문제들도 해결 해가며 진행한 것 같다. 팀원분들 모두 훌륭하신 실력과, 성실한 태도로 임해주셔서 한 분 한 분께 많은 것을 배우고 즐겁게 작업 한 것 같다.
